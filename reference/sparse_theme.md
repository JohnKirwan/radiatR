# Sparse overlay theme for radial plots.

Removes axes, grid lines, and strip decorations to keep attention on the
radial geometry. Useful when adding concentric guides and track
overlays.

## Usage

``` r
sparse_theme(base_theme = ggplot2::theme_classic())
```

## Arguments

- base_theme:

  Base theme to start from. Defaults to \[ggplot2::theme_classic()\].

## Value

A ggplot2 theme object.
