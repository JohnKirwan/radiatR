# Trajectory identifiers of a Tracks

Trajectory identifiers of a Tracks

## Usage

``` r
ids(x)

# S4 method for class 'Tracks'
ids(x)
```

## Arguments

- x:

  A \[Tracks-class\] object.

## Value

The unique trajectory ids, in first-appearance order.
