# Construct a TrajSet from a \*format\* spec (registered name or inline list)

Construct a TrajSet from a \*format\* spec (registered name or inline
list)

## Usage

``` r
TrajSet_read_format(x, format, ...)
```

## Arguments

- x:

  data.frame or path(s)

- format:

  registered format name or list spec

- ...:

  extra args override spec fields
